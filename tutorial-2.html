parte do tutorial 2

<mais tags/>

Agora que você já sabe o que é uma tag, é hora de conhecer mais algumas — só para você reconhecer quando vir. Não precisa decorar nem usar todas ainda.

conceito 1

Novas tags para conhecer

<h1> até <h6> são títulos em ordem de importância, <a> cria um link, <ul> e <li> criam listas. Por enquanto é só para você reconhecer — vamos usá-las de verdade nos próximos tutoriais.

conceito 2

A ordem e a orientação do que cada tag pode fazer

Cada tag nova que você aprende entra dentro da mesma estrutura do tutorial 1: sempre dentro do body, sempre respeitando a ordem em que aparece no código.

<html> <head> <body> <h1>...</h1> <ul>...</ul> </body> </html>

A estrutura da página não muda — só cresce, com mais tags encaixadas na ordem certa.

conceito 3

O resultado do mini tutorial

No próximo tutorial vamos pegar essas tags novas e realmente usá-las para montar uma página com título, lista e link funcionando — com código e resultado lado a lado, como no tutorial 1.

Curiosidade: o <h1> não é só o texto maior da página — ele também ajuda o Google a entender qual é o assunto principal do conteúdo.
