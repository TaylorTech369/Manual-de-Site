parte do tutorial 1

<tags/>

Toda página web é feita de tags. Elas dizem ao navegador o que cada pedaço de conteúdo é — um título, um parágrafo, uma imagem — e como ele deve se comportar.

conceito 1

O que é uma tag e como ela funciona

Uma tag quase sempre vem em par: uma de abertura <p> e uma de fechamento </p>, com o conteúdo no meio. Algumas tags não têm conteúdo, então não precisam de fechamento — como a <img>, que já se fecha sozinha.

conceito 2

A ordem e a orientação do que cada tag pode fazer

Toda página segue uma estrutura básica: <html> envolve tudo, <head> guarda informações que o navegador usa mas não mostra na tela, e <body> é onde fica o conteúdo visível — títulos, textos, imagens. As tags dentro do body aparecem na página na mesma ordem em que são escritas no código.

cartão.html
resultado
<!-- mini exemplo: um cartão de perfil --> <div class="cartao"> <h2>Ana Souza</h2> <p>Aprendendo HTML e CSS 🌱</p> <button>Seguir</button> </div>
conceito 3

O resultado do mini tutorial

O código acima gera o cartão que você vê do lado direito: um título, um texto e um botão, exatamente na ordem em que foram escritos. Troque o texto entre as tags e o resultado muda na mesma hora — é assim que HTML funciona.

Curiosidade: existem mais de 100 tags HTML diferentes, mas no dia a dia você vai usar só umas 20 delas o tempo todo. As outras aparecem aos poucos, conforme a necessidade.
