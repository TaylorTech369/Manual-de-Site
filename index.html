escopo do site

<aprenda/>

Um site de tutoria para quem quer aprender a criar páginas web do zero. Cada tutorial explica um conceito, mostra o código de exemplo e o resultado ao vivo, lado a lado — sem enrolação, no seu ritmo.

como funciona

Um tutorial de cada vez

Navegue pela trilha na barra lateral. Cada aula apresenta um conceito, um mini exemplo prático e uma curiosidade para fixar o que foi aprendido.

o que você vai ver

Código e resultado, lado a lado

Em vez de só olhar uma imagem do código pronto, você vê o código e o resultado renderizado ao vivo, na mesma tela.

o ritmo

Conceitos novos a cada tutorial

Conforme você avança, novos elementos e pequenos exercícios vão sendo apresentados para reforçar o que já foi visto.
